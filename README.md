# Alfred-command-center
🛠️ Getting Started

1)Clone this project using the GitHub link:

-> git clone <clone_url>

2)Install all dependencies:

-> cd <project_folder>

-> npm install

3)create a .env file in the root directory and add required environment variables, for example:

-> MONGO_URI=<url>

-> PORT=<port_number>

4)Run the project:
-> npm run dev
